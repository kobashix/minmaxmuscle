Peptide Scraper Runner

This site is static, so it cannot execute Python directly. Run the scraper locally or on your own server using the command below.

Run the Script

From the repo root, execute:

python scripts/peptide_scraper.py \ --config scripts/peptide_sources.json \ --output-dir generated/peptides \ --format html

What It Does

The bot fetches the URLs listed in scripts/peptide_sources.json, extracts visible text blocks, and generates per-peptide pages in generated/peptides.
